   ```typescript
     loadFavoriteStatus();

   const unsubscribe = subscribeToDataUpdates('favoritesUpdated', () => {
     loadFavoriteStatus();
   });
   ```

3. **useFavorite.ts - saveFavorite调用参数错误** ✅
   ```typescript

     source, id, title, year, ...
   });

   await saveFavorite(source, id, {
     source, id, title, year, ...
   });
   ```
